R & RStudio Installation

The core stack for statistical analysis.

R is the base programming language. You must install this first to process calculations.

Visit CRAN

2. Install RStudio (The Interface)

The Integrated Development Environment (IDE) that provides a user-friendly window for R.

"Download RStudio Desktop (Free) to write, debug, and visualize your code."
Download RStudio

EDA in R 

# 1. Load Data
df <- read.csv("data.csv")

# 2. View basic structure
str(df)
summary(df)

# 3. Check first few rows
head(df)

# 4. Handle Missing Values
df <- na.omit(df)

# 5. Visual EDA (using ggplot2)
library(ggplot2)
ggplot(df, aes(x=variable)) + 
  geom_histogram() + 
  theme_minimal() +
  labs(title="Distribution of Variable")

"Use the Graphical Installer. Check 'Add Anaconda to my PATH' if you want command line access."

EDA in Python 

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# 1. Load Data
df = pd.read_csv('data.csv')

# 2. Inspect Data
print(df.info())
print(df.describe())

# 3. Check for Nulls
print(df.isnull().sum())

# 4. Visual EDA
sns.histplot(df['column_name'])
plt.title('Distribution of Data')
plt.show()

# 5. Correlation Matrix
sns.heatmap(df.corr(), annot=True)
