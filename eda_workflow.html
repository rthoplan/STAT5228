EDA Workflow

Exploratory Data Analysis isn't just about plotting; it's a systematic cycle of questioning and discovery.

1. Question

Define the objective. What are we trying to prove or disprove? Identify the target variable early.

"Is there a correlation between X and Y?"

2. Wrangle

Clean, transform, and reshape. Handle missing values, outliers, and data types to ensure a solid foundation.

df.dropna(), df.pivot_table()

3. Visualise

Uncover distributions and relationships. Use histograms, scatter plots, and heatmaps to "see" the data.

sns.pairplot(), px.histogram()

4. Model

Draft simple baseline models. Check for feature importance and validate the assumptions made during visualization.

LinearRegression(), Logit()

5. Communicate

Synthesize findings. Transform technical results into actionable insights for stakeholders.

Executive Summary, Dashboards

Practical Tips

  • Start Small & Fast Use df.describe() and df.head() before complex functions.
  • Plot Early; Plot Often Visualizing data reveals errors and patterns that tables hide.
  • Annotate Surprises Markdown is your friend. Explain "why" you took a specific cleaning step.
  • Raw Data is Immutable Never overwrite source files. Use df_clean = df.copy().
  • Reproducibility First Save your seed and environment specs to ensure others get your results.
Exploration Script Template 
# --- 1. QUESTION ---
# Goal: Identify factors influencing customer churn

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# --- 2. WRANGLE (Practical Tip: Keep Raw Immutable) ---
df_raw = pd.read_csv('data.csv')
df = df_raw.copy()

# Fast Summaries
print(df.info())
print(df.describe())

# Clean
df['total_spend'] = df['total_spend'].fillna(df['total_spend'].median())

# --- 3. VISUALISE (Practical Tip: Plot Early) ---
plt.figure(figsize=(10, 6))
sns.heatmap(df.corr(), annot=True, cmap='coolwarm')
plt.title("Correlation Matrix - Initial Insight")
plt.show()
