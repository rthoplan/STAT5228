System Check

  • Windows 10/11 (64-bit)
  • No Native macOS app: Mac users require a VM (Parallels) or Boot Camp.
  • Min. 4GB RAM (8GB+ recommended)

Do you need an account?

NO

To build reports locally (PBIX files). You can install and use the software for the whole semester offline.

YES

To publish reports to the cloud (PBI Service) or collaborate with others via the browser.

Power BI "Free" Capability Matrix

What You CAN Do

  • Connect to hundreds of data sources (Excel, Web, SQL).
  • Full Power Query (cleaning) and DAX (formulas) features.
  • Export to PDF or PowerPoint.
  • Create your own "Personal Sandbox" in the cloud.

What is Restricted

  • Sharing reports with classmates (requires Pro/Premium).
  • Consuming content shared by others (unless in Premium Workspace).
  • Automated data refresh schedules are limited.

Warning: Publish to Web

"Publish to web" (Public) is NOT the same as "Publishing to Service".

Microsoft warns that the "Publish to web" feature creates a link that anyone on the internet can access. This means:

No login required to see the data.
Reports can be indexed by search engines.
All underlying data could potentially be accessible.

Best Practice:

"For our class, keep your data in the PBI Desktop (PBIX) file unless specifically instructed otherwise. Never use 'Publish to Web' with confidential or personal student data."
